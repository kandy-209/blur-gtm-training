## What You Need to Do Next

### Step 1: Add Domain in Vercel (2 minutes)

1. **Go to:** https://vercel.com/dashboard
2. **Click:** `cursor-gtm-training` project
3. **Settings** → **Domains**
4. **Click:** **Add Domain**
5. **Enter:** `cursor-gtm-enablement.com`
6. **Click:** **Add**

### Step 2: Configure DNS (2 minutes)

**At your domain registrar** (where you bought `cursor-gtm-enablement.com`):

**Add CNAME Record:**
```
Type: CNAME
Name: @ (or root/blank)
Value: cname.vercel-dns.com
TTL: 3600
```

**OR A Record:**
```
Type: A
Name: @
Value: 76.76.21.21
TTL: 3600
```

- **Wait:** 1-2 hours for DNS propagation
- **Check:** Vercel dashboard shows "Valid"
